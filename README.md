# 🤖 Rokey_isaac-sim
> **두산루키 아이작심 프로젝트 (Doosan Rokey Isaac Sim Project)**

본 프로젝트는 NVIDIA Isaac Sim 환경 내에서 협동로봇 시뮬레이션을 구현하고 제어하기 위한 레포지토리입니다.

---

## 📌 주요 스크립트 역할

| 파일명 | 담당 역할 |
| :--- | :--- |
| **`main.py`** | 시스템의 메인 컨트롤러로, **모든 파일의 실행**을 담당합니다. |
| **`robot_config.py`** | 시뮬레이션 환경 내 **로봇 스폰(Spawn)**을 담당합니다. |
| **`world_setup.py`** | **환경 구성**(맵 로드 및 조명 설정)을 담당합니다. |

---

## 📂 디렉토리 구조 및 관리 가이드

프로젝트의 유지보수를 위해 아래의 폴더 구조 규칙을 반드시 준수해 주세요.

### 1. `usd/` 폴더
* 불러와야 할 **USD 파일만** 깔끔하게 남겨두고 관리합니다.

### 2. `robots/` 폴더
* 실제 사용하는 로봇별로 개별 폴더를 생성합니다.
* 각 로봇 폴더 안의 메인 스크립트 명은 **`{로봇이름}_agent.py`** 형식으로 통일합니다.
* 구조는 반드시 아래와 같은 형태로 정리되어야 합니다:

```text
robots/
└── {본인_로봇_폴더}/
    └── {로봇이름}_agent.py  <- 메인 스크립트


## 🚀 실행 방법 (How to Run)

우분투 터미널을 열고 Isaac Sim의 고유 파이썬 환경(`isaac-python`)을 이용하여 메인 스크립트를 실행합니다.

### 1. 레포지토리 디렉토리로 이동
```bash
cd /home/rokey/Rokey_isaac-sim

isaac-python main_isaac/main.py
```
