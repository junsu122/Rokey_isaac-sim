## 초기 상태 (시뮬 시작 시)

- **full pod: 9개** (pod_12 ~ pod_20) — 구역 안쪽(벽 쪽)에 배치
- **empty pod: 11개** (pod_01 ~ pod_11) — 구역 바깥쪽(통로 쪽)에 배치
- iw_hub는 empty pod 중 하나(pod_01)를 먼저 픽업해 컨베이어 앞으로 이동

### Pod 번호 배정 (행 순서, 왼쪽→오른쪽)

```
행\열   1열(-2.25)  2열(-0.75)  3열(0.75)  4열(2.25)
1행     pod_01      pod_02      pod_03     pod_04
2행     pod_05      pod_06      pod_07     pod_08
3행     pod_09      pod_10      pod_11     pod_12  ← full 시작
4행     pod_13      pod_14      pod_15     pod_16
5행     pod_17      pod_18      pod_19     pod_20
```

| pod_id | 초기 state |
|--------|-----------|
| pod_01 ~ pod_11 | empty |
| pod_12 ~ pod_20 | full |

## robot_config.py 적용 방식 (예시)

```python
def _gen_pod_coords(y_rows):
    """4열 × 5행 격자 좌표 생성"""
    xs = [-2.25, -0.75, 0.75, 2.25]
    coords = []
    for y in y_rows:
        for x in xs:
            coords.append((x, y, 0.0))
    return coords

SECTOR_POD_LAYOUT = {
    "A": {
        "y_rows": [7.3, 8.7, 10.1, 11.5, 12.9],
        "full_count": 9,
    },
    "B": {
        "y_rows": [-2.8, -1.4, 0.0, 1.4, 2.8],
        "full_count": 9,
    },
    "C": {
        "y_rows": [-7.3, -8.7, -10.1, -11.5, -12.9],
        "full_count": 9,
    },
}
```
